    1. Parentheses (,)
    2. connectives $\lor,\neg$
    3. Quantifier $\forall$
    4. variables, one for each positive integer n:
       $v_1, v_2, \dots, v_n$
       Often denoted $Vars$
    5. Equality symbol : =
    6. constant symbols
    7. function symbols
    8. relation symbols

    2. $t$ is a constant symbol
    3. $t:=ft_1t_2\dots t_n$, where $f$ is an n-ary function symbol
